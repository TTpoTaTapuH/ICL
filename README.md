## Contributing
#### Installation
1.`install python 3.7 and docker`
2. `pip3 install -r requirements.txt`
3. `make run-postgres`
4. `make migrate`
5. Создаем пользователя для админки: `make createsuperuser`

#### Endpoints
- Admin page: http://localhost:8060/admin/
- get script http://localhost:8060/api/task/
